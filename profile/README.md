## Hi there 👋

> 이사를 할 때, 전세, 월세를 알아볼 때, 절차에 따라서 해야할 것들이 많아 체크리스트를 적어 진행하면 좋을 것 같아 기획한 서비스입니다. 중요한 조건들을 놓치지 않도록 필수적인 것들은 기본으로 설정해두고, 사용자가 원하는 항목을 추가할 수 있는 웹 사이트입니다.  
**이사할 때 친구처럼 도와준다는 의미로 Lisa + Mate** 
>
> 

<br><br>

## ✨ 주요 기능
🔐 **회원가입/로그인**

: 닉네임, 이메일, 비밀번호 입력으로 간단한 회원가입으로 로그인이 가능합니다!  

**🧑‍🏫 게시판 CRUD**

: 정보 게시판, 후기 게시판, 우리동네 게시판 등 다양한 게시판으로 이사할 때 정보를 얻어요!

✅ **체크리스트**

: 날짜와 기간(이사전, 이사중, 이사후)를 정하여 해야하는 일을 기록할 수 있어요!

→ 메인페이지와 달력에서 간편하게 확인할 수 있어요!

😎 **프로필 편집**

: 닉네임, 소개글과 프로필 사진을 수정할 수 있어요!
